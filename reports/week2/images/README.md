# Images Directory

This directory contains screenshots and images for the Assignment 2 report.

## Required Images

Please add the following images before submission:

1. **protected-branch-settings.png**
   - Screenshot of GitHub/GitLab branch protection settings
   - Show that direct pushes are disabled
   - Show that approvals are required

2. **example-reviewed-pr.png**
   - Screenshot of an example reviewed Pull Request
   - Must show review by another team member (not self-review)
   - Include comments and approval status

3. **figma-prototype.png**
   - Screenshot of the Figma prototype
   - Show key screens and navigation
   - Include important states (empty, success, error)

4. **mvp-v0-deployment.png**
   - Screenshot of the deployed MVP v0
   - Show the application running
   - Include URL in browser address bar

## Image Guidelines

- **Format:** PNG (preferred) or JPG
- **Resolution:** At least 1920x1080 for desktop, 375x812 for mobile
- **File Size:** Keep under 2MB per image
- **Naming:** Use lowercase with hyphens (kebab-case)
- **Privacy:** Sanitize any personal information, real names, or sensitive data

## How to Take Screenshots

### macOS
- Full screen: `Cmd + Shift + 3`
- Selected area: `Cmd + Shift + 4`
- Window: `Cmd + Shift + 4` then `Space`

### Windows
- Full screen: `Print Screen`
- Selected area: `Win + Shift + S`
- Window: `Alt + Print Screen`

### Linux
- Full screen: `Print Screen`
- Selected area: `Shift + Print Screen`

## Adding Images

1. Take the screenshot following the guidelines above
2. Rename the file to match the required filename
3. Place the file in this directory
4. Update `reports/week2/README.md` if needed
