# LLM Usage Report - BasketForm-AI

## Overview

This report documents the use of AI/LLM tools throughout Assignment 2 for the BasketForm-AI project. All AI/LLM usage has been disclosed and documented as per course requirements.

## Tools Used

### 1. GitHub Copilot
**Usage Period:** Throughout Assignment 2
**Primary Use:** Code completion and generation

**Specific Applications:**
- Generated boilerplate code for React components
- Assisted with API endpoint definitions
- Provided code suggestions for video upload functionality
- Helped with database schema design

**Impact:** Increased development speed by approximately 30% for routine coding tasks.

### 2. ChatGPT (GPT-4)
**Usage Period:** During planning and documentation phases
**Primary Use:** Research, brainstorming, and document drafting

**Specific Applications:**
- **User Story Development:** Generated initial user story drafts based on project requirements
- **MoSCoW Prioritization:** Provided framework for prioritization discussions
- **Technical Research:** Researched biomechanical analysis libraries and approaches
- **Documentation:** Assisted with report writing and formatting

**Impact:** Accelerated research and documentation phases, provided diverse perspectives for user story development.

### 3. Claude (Anthropic)
**Usage Period:** During analysis and review phases
**Primary Use:** Code review, analysis, and feedback

**Specific Applications:**
- **Code Review:** Reviewed pull requests and provided suggestions
- **Architecture Analysis:** Analyzed technical architecture decisions
- **Feedback Integration:** Helped process customer feedback into actionable items
- **Risk Assessment:** Assisted with identifying and mitigating project risks

**Impact:** Improved code quality and provided balanced analysis of technical decisions.

### 4. Figma AI Features
**Usage Period:** During prototyping phase
**Primary Use:** Design assistance and iteration

**Specific Applications:**
- **Layout Suggestions:** Generated initial layout options for key screens
- **Color Palette Generation:** Suggested color schemes based on sports/basketball theme
- **Component Variations:** Generated multiple variations of UI components
- **Responsive Design:** Assisted with mobile-first design approach

**Impact:** Accelerated prototyping phase and provided design alternatives for consideration.

## Ethical Considerations

### Transparency
- All AI usage has been documented in this report
- Customer was informed about AI assistance during meeting
- Team members are aware of AI limitations and potential biases

### Originality
- AI was used as a tool to assist, not replace, human creativity
- Final decisions were made by human team members
- AI suggestions were adapted and customized to project needs

### Attribution
- AI tools were used with proper licensing and terms of service
- No proprietary information was shared with AI tools
- Customer data was not processed through AI tools

## Lessons Learned

### Effective AI Usage
1. **Clear prompts produce better results:** Well-defined prompts for AI tools generated more useful outputs
2. **Human oversight is essential:** All AI outputs required human review and modification
3. **Iterative refinement:** Multiple iterations with AI tools improved final quality
4. **Domain knowledge matters:** Understanding the domain helped evaluate AI suggestions

### Challenges Encountered
1. **AI limitations:** Some complex technical requirements exceeded AI capabilities
2. **Context understanding:** AI sometimes missed project-specific context
3. **Consistency:** Maintaining consistent style required human intervention
4. **Verification:** AI outputs required verification against actual requirements

## Recommendations for Future Assignments

1. **Establish AI guidelines early:** Define clear guidelines for AI usage in team workflow
2. **Integrate AI review processes:** Include AI-generated content in regular review cycles
3. **Track AI impact:** Measure time savings and quality improvements
4. **Balance AI and human work:** Maintain appropriate balance for learning and skill development

## Conclusion

AI/LLM tools were used effectively throughout Assignment 2 to accelerate development, improve documentation quality, and provide diverse perspectives. All usage was transparent, ethical, and complemented human team members' work rather than replacing it. The tools assisted with routine tasks, research, and initial drafts, while humans made all final decisions and maintained creative control.

The team maintained appropriate balance between AI assistance and human contribution, ensuring that learning objectives were met while leveraging AI capabilities for efficiency gains.
